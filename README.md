# PATH_TO_CLONE_NEW_REPO

1º- Criar novo repsitório no GitHub;
2º- Copiar link HTTPS na Aba Code;
3º- Abrir o VsCode + New Window;
4º- Clicar: 'Clone Git Repository';
5º- Colar o link HTTPS + Enter;
6º- Selecionar Pasta Local;
7º- Clicar em Open para abrir o VsCode;
8º- O VsCode abre com Readme na raiz do repositório;
